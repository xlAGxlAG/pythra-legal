Privacy Policy

Effective Date: January 30, 2026
Last Updated: January 30, 2026

PolymAI (“the Extension”) is a Chrome browser extension operated by Pallax. We respect your privacy and are committed to protecting user data. This Privacy Policy explains how information is collected, used, and stored when you use the Extension.

1. Information We Do Not Collect

The Extension does not:

  • Collect Polymarket account passwords
  • Access or store wallet private keys, seed phrases, or credentials
  • Collect payment information
  • Track browsing activity outside supported pages
  • Sell, rent, or share personal data for advertising purposes

2. Information We May Collect

The Extension collects only the minimum data necessary to provide its functionality.

2.1 Local Storage Data

The following data may be stored locally in the user’s browser:

  • User preferences and settings
  • Interface state and layout configuration
  • Locally stored interaction or analysis history (if enabled)

This data is stored locally on the user’s device and is not automatically transmitted to Pallax servers.

2.2 Optional Account Information

If the Extension provides optional sign-in features in the future, basic account identifiers such as an email address may be used solely for authentication and preference synchronization.

3. Permissions Used by the Extension

The Extension only requests the following permissions:

  • storage: Used to store user preferences and settings locally.
  • unlimitedStorage: Used to allow sufficient local storage for user data and historical records.
  • sidePanel: Used to display and interact with the Extension’s interface within the browser side panel.

The Extension does not use the scripting permission and does not inject or execute scripts on web pages.

4. Purpose of Data Usage

Collected data is used only to:

  • Provide the core functionality of the Extension
  • Maintain user preferences and settings
  • Improve usability and performance

5. Data Storage and Security

  • Data is primarily stored locally within the browser
  • Reasonable technical measures are used to protect stored data
  • No unnecessary data is retained beyond its functional purpose

6. Data Sharing

Pallax does not sell or share personal data with third parties. Data is not used for advertising or tracking purposes.

7. User Control

Users may at any time:

  • Disable or remove the Extension via browser settings
  • Clear locally stored data through their browser

Removing the Extension deletes all locally stored data.

8. Children’s Privacy

The Extension is not intended for children under the age of 13. We do not knowingly collect personal data from children.

9. Changes to This Policy

This Privacy Policy may be updated from time to time. Updates will be reflected on this page with a revised “Last Updated” date.
