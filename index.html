Privacy Policy

Effective Date: January 30, 2026
Last Updated: January 30, 2026

PolymAI (“the Extension”) is a Chrome browser extension operated by Pallax. This Privacy Policy explains how information is collected, used, and stored when you use the Extension.

1. Information We Do Not Collect

The Extension does not:

  • Collect account passwords or authentication credentials
  • Access or store wallet private keys or seed phrases
  • Collect payment or financial information
  • Sell or share personal data for advertising purposes

2. Information We May Collect

The Extension collects only the minimum information required to provide its functionality.

2.1 Local Storage Data

The following data may be stored locally in the user’s browser:

  • User preferences and settings
  • Interface state and layout configuration
  • Locally stored historical or analytical data generated by the Extension

This data is stored locally on the user’s device and is not automatically transmitted to Pallax servers.

3. Permissions Used by the Extension

The Extension only requests permissions necessary for its core functionality:

  • storage: Used to save user preferences and configuration locally.
  • unlimitedStorage: Used to allow sufficient local storage for user data and historical records.
  • sidePanel: Used to display and interact with the Extension’s interface within the browser side panel.

4. Purpose of Data Usage

Data is used solely to:

  • Provide the Extension’s features
  • Maintain user preferences and settings
  • Improve usability and performance

5. Data Storage and Security

  • Data is primarily stored locally within the browser
  • Reasonable technical measures are used to protect stored data
  • Data is retained only as long as necessary for functionality

6. Data Sharing

Pallax does not sell or rent personal data to third parties. Data is not used for advertising or tracking purposes.

7. User Control

Users may at any time:

  • Disable or remove the Extension through browser settings
  • Clear locally stored data using browser controls

Removing the Extension deletes all locally stored data.

8. Children’s Privacy

The Extension is not intended for children under the age of 13. No personal data from children is knowingly collected.

9. Changes to This Policy

This Privacy Policy may be updated periodically. Any updates will be reflected on this page with a revised date.
